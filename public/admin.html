Модель ИИ

Искусственный интеллект отправляет данные из заявки.

При составлении промпта используйте якоря из списка ниже.

Они будут заменены на информацию из заявки:

${workers} - Имя исполнителя

${device} - Тип устройства

${model} - Модель устройства

${crush} - Неисправность

${note} - Информация из доп. поля
